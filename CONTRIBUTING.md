# Contributing to Exam Preparation AI Agent

Thank you for your interest in contributing!

## How to Contribute

### 1. Folder Structure Adherence
Please respect the repository structure:
- **`software/agent/`**: All source code for the AI agent must be placed here.

### 2. Submitting Changes
1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/your-feature-name`).
3. Make your changes and test the relevant subsystem.
4. Open a Pull Request with a clear description.
